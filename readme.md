Built with:
* Node.js
* Express
* Handlebars
* Postgres/Sequelize

## Installation
1. Clone the repo
`git clone https://github.com/reikamoon/cutie_smoothies.git`
2. In the repo folder, install necessary ingredients (NPM packages)
`npm i`
3. Run nodemon
`nodemon app.js`
